The Kitchen Edition — AI Meal Suite

This repository is a scaffold for The Kitchen Edition: an editorial magazine-styled
AI meal planning app built with React + TypeScript + Vite + Tailwind.

Quick start

1. Copy `.env.example` to `.env` and populate values for Firebase, Supabase, Spoonacular, and Stripe.
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev` (Vite on port 5173)

Notes

- Firebase is used for realtime weekly meal plans.
- Supabase is used for auth and the `subscriptions` table.
- Spoonacular is used for recipe data; add your API key to the env.
